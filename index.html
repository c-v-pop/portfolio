Groupomania App

The Groupomania App is a full-stack social media application built from scratch using the MERN stack (MongoDB, Express, ReactJS, NodeJS). The app allows users to create accounts, upload photos, and interact with posts through comments. This was a CRUD application designed for business use, where employees can interact with each other’s posts and share images. I integrated Multer for file uploads, Bcrypt for password encryption, jsonwebtoken for authentication, and used MySQL for the database. This project helped me strengthen my skills in full-stack development, authentication, and file management. Don't forget to check out the GitHub repository for more details by clicking on the image!

Laravel To-Do List

This Laravel To-Do List project was part of a technical test given by a potential employer. The app was built using Laravel as the PHP framework, MySQL for the database, and Tailwind CSS for styling. I structured the app with MVC architecture for scalability and efficient task management. MySQL was used for task storage, taking advantage of its relational database capabilities. The backend logic was built in PHP, ensuring smooth task management functionality. The responsive UI, styled with Tailwind CSS, provided a polished and user-friendly experience.
